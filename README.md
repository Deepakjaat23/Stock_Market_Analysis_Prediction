# 📈 Stock Market Analysis & Prediction

A Python-based Stock Market Analysis and Prediction system that allows users to analyze historical stock data, visualize trends, calculate technical indicators, and predict future stock prices using Machine Learning.

## 🚀 Features

* 📊 Fetch historical stock market data
* 📈 Interactive data visualization
* 🔍 Exploratory Data Analysis (EDA)
* 📉 Technical Indicators

  * Moving Average (MA)
  * Exponential Moving Average (EMA)
  * Relative Strength Index (RSI)
  * MACD
  * Bollinger Bands
* 🤖 Stock Price Prediction using Machine Learning
* 📅 Predict future stock prices based on historical trends
* 💻 Menu-driven command line interface

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Deepakjaat23/Stock_Market_Analysis_Prediction.git
```

### 2. Move into the project folder

```bash
cd Stock_Market_Analysis_Prediction
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python main.py
```
